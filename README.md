# Vue.js 3 + Vite + TypeScript 实战项目开发

本模块中我们通过使用 TypeScript 编程语言，基于 Vue.js 3 全家桶（Vue.js、Vue Router、Vuex、Vite、Element Plus 等）开发 B 端管理系统项目（dashboard)。通过实战深入掌握 Vue.js 3 及其相关技术栈的使用。

## 任务

- 任务一：搭建项目架构（上）
- 任务二：搭建项目架构（下）
- 任务三：用户登录和身份认证
- 任务四：权限管理
- 任务五：商品管理
- 任务六：发布部署
